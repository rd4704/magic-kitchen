# Magic Kitchen 🌟

A fun cooking game for kids where cute animal customers order treats!

## How to Play

1. A cute animal customer appears and asks for a specific treat
2. Tap 2 ingredients from the shelf to add them to the mixing bowl
3. Press ✨ Mix! to craft the item
4. If it's what the customer wants → they celebrate and you get a ⭐
5. Stuck? Hit the 💡 Hint button!

## Recipes

8 secret recipes to discover by combining ingredients like Flour, Egg, Berry, Choco, and more — making cookies, donuts, muffins, lemonade, and even a Magic Potion! ✨

The game tracks all the recipes your child finds in a little recipe book as they play.

## Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```

## Deployment

The `dist` folder after `npm run build` can be deployed to any static hosting service (Vercel, Netlify, GitHub Pages, etc.)
